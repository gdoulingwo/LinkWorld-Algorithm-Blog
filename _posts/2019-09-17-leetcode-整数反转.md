# 代码实现

## Python实现

```python
class Solution:
    def reverse(self, x: int) -> int:
        reverse_x = int(str(x)[::-1].rstrip('-'))
        if reverse_x < pow(2, 31):
            if x >= 0:
                return reverse_x
            else:
                return 0 - reverse_x
        return 0
```
